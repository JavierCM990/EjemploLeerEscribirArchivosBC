# Ejemplo Leer y Escribir Archivos BC

Ejercicio de Business Central desarrollado en AL.

## Descripción
Extensión que permite cargar imágenes para un artículo desde la 
ficha de artículo en Business Central.

## Archivos
- **ImageManagement.CodeUnit.al** - Codeunit con el procedimiento para importar imágenes
- **ItemCardPictureExtension.PageExt.al** - Extensión de la página Ficha de artículo con la acción de importar

## Configuración
- Runtime: 16.1
- Application: 27.0.0.0
